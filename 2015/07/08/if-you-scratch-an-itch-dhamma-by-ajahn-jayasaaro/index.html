If you scratch an itch ... - Dhamma by Ajahn Jayasāro

July 08, 2015
Edit on Github

If you scratch an itch it feels good, and you feel better for a while. But after some time the itch comes back, usually stronger than before. The more you scratch, the more you itch.

Satisfying our cravings is like scratching an itch. The Buddha encouraged us to inquire: Which is better: the pleasure and temporary relief that comes from scratching an itch, or the happiness of having no itch to scratch?

Ajahn Jayasāro
