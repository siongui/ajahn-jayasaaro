พฤษภาคม 03, 2017
การแปล: English 
แก้ไขที่ Github

บางคนเชื่อว่ามีสวรรค์หรือนรกชั่วนิรันดร์ บางคนเชื่อว่าตายแล้วทุกอย่างดับสูญ บางคนเชื่อว่าอย่าคิดถึงความตายและความหมายของคำว่าตายจะดีกว่า

พระพุทธศาสนาสอนว่ายิ่งเราเข้าใจหลักธรรมที่ว่าทุกสิ่งเป็นไปตามเหตุและปัจจัยและเข้าใจความจริงของกายกับใจ ความเชื่อเรื่องการดับสูญหรือการมีชีวิตนิรันดร์ย่อมจะลดลง ให้เราเฝ้าดูจิตใจของตนอย่างละเอียด แล้วมรรคอันเป็นทางสายกลางจะค่อยๆ ปรากฏขึ้น
