กรกฎาคม 12, 2017
การแปล: English 
แก้ไขที่ Github

การรักษาใจให้จดจ่ออยู่ในอารมณ์กรรมฐานไม่ได้อยู่ที่การห้ามใจไม่ให้หลงเพลินไปกับความคิดรบกวนเพียงอย่างเดียวเท่านั้น แต่ยังขึ้นอยู่กับการสร้างความรู้สึกที่ดีต่ออารมณ์กรรมฐานนั้นด้วย เมื่อใดที่เรารู้สึกพอใจในความเรียบง่ายและความละเอียดอ่อนของลมหายใจจนไม่ต้องเที่ยวหาอารมณ์อื่นใดมาสร้างความบันเทิงให้เรา เมื่อนั้นย่อมเป็นการเรียนรู้ศิลปะแห่งความสันโดษ

หากเราสร้างความพอใจกับลมหายใจได้ เราย่อมสร้างความพอใจในวัตถุข้าวของที่เราครอบครองในทางโลกเช่นกัน ความสันโดษจึงหมายถึงการตระหนักในคุณค่าของสิ่งที่เรามีอยู่ในปัจจุบันขณะ
