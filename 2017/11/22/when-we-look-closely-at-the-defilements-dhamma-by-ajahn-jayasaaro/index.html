When we look closely at the defilements ... - Dhamma by Ajahn Jayasāro

November 22, 2017
Edit on Github

When we look closely at the defilements that we are still unable to abandon, we find that there is always something about them that we enjoy and don't want to let go of. It is only when we recognise the pleasurable sensations and sense of self that defilements give us, and are willing to renounce them, that inner freedom can be experienced.

Ajahn Jayasāro
