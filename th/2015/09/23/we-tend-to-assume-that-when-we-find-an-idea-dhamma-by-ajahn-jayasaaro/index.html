เวลาเจอคำสอนที่เข้าใจได้ยาก ... - ธรรมะ โดย พระอาจารย์ชยสาโร

กันยายน 23, 2015
การแปล: English 
แก้ไขที่ Github

เวลาเจอคำสอนที่เข้าใจได้ยาก เรามักจะสันนิษฐานไปว่าเรื่องนี้คงต้องลึกซึ้งมาก แต่แท้จริงแล้ว ข้อคิดที่อธิบายด้วยศัพท์แสงยืดยาวและยากโดยไม่จำเป็นนั้นกลับตื้นเขินอยู่บ่อยครั้ง ภาษาถูกใช้เป็นเครื่องมืออำพรางความคิดมากกว่าจะเปิดเผยความจริง และใช้ข่มให้ลูกศิษย์ครั่นคร้าม

เรื่องที่ลึกซึ้งที่สุดนั้นไม่ได้ยากเกินจะคิดทำความเข้าใจ อะไรเล่าจะอธิบายได้ง่ายดายกว่า 'ความไม่แน่’

สิ่งท้าทายจึงอยู่ที่การฝึกฝนจิตใจ ให้ถึงขั้นที่ความเข้าใจเชิงเหตุผลนั้นจะทำหน้าที่เป็นแท่นกระโดด ส่งให้จิตทะยานเข้าถึงความรู้แจ้งในระดับเปลี่ยนชีวิตได้ ข้ามพ้นกับดักแห่งถ้อยคำใดๆ
