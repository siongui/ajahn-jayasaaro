แก้ไขที่ Github

หากมีคนวิจารณ์เราอย่างไม่เป็นธรรมด้วยเจตนาร้าย พึงตั้งสติ หายใจเข้าออกให้เป็นปกติ และรักษาใจไม่ให้โกรธแค้น

หากมีคนวิจารณ์เราอย่างสมควรแต่ด้วยเจตนาร้าย พึงมองข้ามเจตนาร้ายนั้น และรับเฉพาะข้อมูลด้วยสำนึกในคุณว่าเป็นโอกาสให้แก้ไขข้อผิดพลาด

หากมีคนวิจารณ์เราอย่างไม่เป็นธรรมด้วยความปรารถนาดี พึงดูแต่ความปรารถนาดีภายใต้ถ้อยคำอันระคายหูนั้น

และเมื่อใดที่มีคนวิจารณ์เราอย่างสมควรด้วยความปรารถนาดี พึงรับฟังอย่างตั้งใจและยินดียิ่งที่มีกัลยาณมิตรพร้อมจะชี้แนะข้อบกพร่องที่เราไม่อาจมองเห็น
