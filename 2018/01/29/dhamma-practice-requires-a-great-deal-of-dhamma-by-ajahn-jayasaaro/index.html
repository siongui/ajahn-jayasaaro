Dhamma practice requires a great deal of ... - Dhamma by Ajahn Jayasāro

January 29, 2018
Translation(s): ไทย 
Edit on Github

Dhamma practice requires a great deal of trial and error. We learn best by trying things out and seeing what works. Each error gives us important information that we can use to improve the quality of our effort. If we can recognize errors as useful feedback rather than as proof that we are stupid or full of defilements, we will swiftly progress.

Ajahn Jayasāro
